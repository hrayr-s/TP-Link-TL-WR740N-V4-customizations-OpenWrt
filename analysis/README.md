# Analysis Artifacts

`analysis/report.md` is the reusable analysis summary for the two original
TP-Link firmware images that were inspected during this project.

The raw OEM firmware images, extracted kernels, SquashFS root filesystems, and
generated file inventories are intentionally ignored by Git. They can contain
copyrighted vendor firmware contents and should be regenerated locally when
needed rather than committed to a public repository.
